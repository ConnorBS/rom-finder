## Name
ROM Finder

## Short description (132 chars max)
Add RetroAchievements games to your Wanted list and search ROM sources directly from any game page.

## Detailed description
ROM Finder connects your browser to your self-hosted ROM Finder instance, so you can manage your game collection without leaving RetroAchievements.

**Features**
• One-click "Add to Wanted" — adds the current game to your ROM Finder Wanted list straight from the RetroAchievements game page
• Search ROM sources — query your enabled sources (Internet Archive, VIMM's Lair, and more) inline, without switching tabs
• Auto-detection — the extension reads the game title, system, and RA game ID directly from the page
• Floating panel — unobtrusive pill button sits in the corner of every retroachievements.org/game/… page; expand when you need it
• Configurable server — works with localhost, a home network IP, or a Docker deployment on your server

**Requirements**
A running instance of ROM Finder (free, open source, self-hosted). Set your server address once in the extension Options page. Defaults to http://127.0.0.1:8080.

**Privacy**
This extension only communicates with your own self-hosted ROM Finder server. No data is collected or sent to any third party.
