ROM Finder

Configure your local ROM Finder instance.

The address where ROM Finder is running. Include the port number.

Local: http://127.0.0.1:8080 (default)
Docker / server: use the IP or domain where you're hosting ROM Finder, e.g. http://192.168.1.100:8080 or http://romfinder.home.

Make sure CORS is enabled on the server (it is by default).
